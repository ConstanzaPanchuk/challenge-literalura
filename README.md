<h1 align="center"> Challenge Alura – LiterAlura :books:</h1>

:notebook: **Descripción del proyecto**

Proyecto desarrollado en Java con Spring Boot y PostgreSQL que permite gestionar información de libros y autores.  
El sistema permite registrar libros y autores en la base de datos, realizar búsquedas por título de libro o por autor, listar autores vivos en un año específico y filtrar libros por idioma.  
La aplicación se ejecuta en consola y utiliza Spring Data JPA para la persistencia de datos, lo que facilita el manejo de consultas y estadísticas básicas sobre los libros y autores registrados.

:hammer:**Funcionalidades del proyecto**
- Buscar libros por título y registrar autores en la base de datos.
  
- Listar libros y autores registrados.
  
- Mostrar autores vivos en un año determinado.
  
- Filtrar libros por idioma.
  
- Posibilidad de agregar nuevas consultas y estadísticas avanzadas.
